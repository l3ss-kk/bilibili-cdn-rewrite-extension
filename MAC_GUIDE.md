# Mac 新手使用说明

这份说明只讲怎么把插件装到 Mac 的 Chrome 里测试。

## 你会拿到什么

你会拿到一个压缩包。先把它解压，解压后会得到一个插件文件夹。

不要打开里面的代码文件，也不要改文件名。

## 安装步骤

1. 打开 Mac 上的 Chrome 浏览器。
2. 在地址栏输入：

```text
chrome://extensions/
```

3. 按回车。
4. 打开右上角的 **开发者模式**。
5. 点击 **加载已解压的扩展程序**。
6. 选择刚才解压出来的插件文件夹。
7. 如果页面里出现 **B站视频线路切换**，说明安装成功。

## 怎么用

1. 打开一个 B 站视频或直播回放。
2. 点击 Chrome 右上角的扩展图标。
3. 点 **B站视频线路切换**。
4. 选择一条线路，比如 `mirrorhw`。
5. 页面会自动刷新；如果没有刷新，就手动刷新一次。
6. 看视频是否还频繁缓冲。

## 怎么换线路

点扩展图标，再选择另一条线路即可。

可以按这个顺序试：

1. `mirrorhw`
2. `mirrorali`
3. `mirroralib`
4. `mirrorhwb`

哪条最顺就用哪条。

## 怎么关闭

回到：

```text
chrome://extensions/
```

找到 **B站视频线路切换**，把开关关掉即可。

不想用了，也可以点 **移除**。
